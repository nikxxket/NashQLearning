# Nash Q-Learning для координации агентов в сеточном мире

## О проекте:

Реализация алгоритма **Nash Q‑Learning** для обучения двух агентов в дискретной среде с сеточным миром.  
Проект демонстрирует применение теории игр (равновесие Нэша) в многоагентном обучении с подкреплением для решения задач координации.

## Постановка задачи:

### Правила среды
- **Поле:** сетка `3×3`, ячейки пронумерованы от `0` до `8`.
- **Агенты:** два игрока, стартовые позиции — `0` и `2`.
- **Действия:** `UP`, `DOWN`, `LEFT`, `RIGHT`.
- **Цели:**
  - **Игра 1:** Агент 1 → ячейка `8`, Агент 2 → ячейка `6`.  
    Вход в цель разрешён только снизу (в `8` из `5`, в `6` из `3`).
  - **Игра 2:** Общая цель — ячейка `7` для обоих агентов.
- **Столкновения:** при попытке занять одну и ту же ячейку (кроме целей) агенты отскакивают и получают штраф `-1`.
- **Награды:**
  - Достижение цели: `+100`
  - Столкновение: `-1`
  - Шаговый штраф (только для активных агентов): `-0.05`
  - Штраф за бездействие: `-0.1`
- **Режимы:**
  - Детерминированный — переходы выполняются всегда.
  - Стохастический — из ячеек `0` и `2` действие `UP` срабатывает с вероятностью `0.5`.

### Особенности обучения
- Агенты **не знают** расположение целей, награды друг друга и динамику среды.
- На каждом шаге наблюдают совместное состояние `(pos1, pos2)` и получают индивидуальные награды.
- Требуется найти равновесие Нэша в стационарных стратегиях.

## 🧠 Алгоритм Nash Q‑Learning

### Формализация
Пространство состояний:  
`S = { (0,0), (0,1), …, (8,7) }` — всего `81` состояние.

Для каждого состояния строится биматричная игра с матрицами выигрышей `Q₁(s)` и `Q₂(s)` размера `4×4`.  
Обновление Q‑значений происходит по формуле:
Qᵢ(s, a₁, a₂) ← Qᵢ(s, a₁, a₂) + α [ rᵢ + γ · NashValueᵢ(s') - Qᵢ(s, a₁, a₂) ]

text

где `NashValueᵢ(s')` — ожидаемый выигрыш агента `i` в равновесии Нэша для следующего состояния.

### Поиск равновесия Нэша
1. Проверка **чистых** равновесий (пары действий, от которых невыгодно отклоняться ни одному агенту).
2. Если чистых нет — вычисляется **смешанное** равновесие с помощью minimax стратегии (упрощённый подход).
